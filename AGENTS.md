# AGENTS.md

## 项目目标
本项目是一个 AI 工具选型 SEO 内容站，用于通过搜索流量引导用户进入独立小店购买。小店地址：https://pay.ldxp.cn/shop/ai007

## 核心要求
- 使用静态 HTML 或服务端渲染方式输出正文。
- 不使用纯客户端 JS 承载关键正文内容。
- 不做自动跳转、隐藏文字、关键词堆砌、门页、批量低质页面。
- 不复制小店全部商品内容。
- 内容站负责搜索承接和选型教育，小店负责交易。
- 所有购买入口必须是用户主动点击。
- 所有购买入口使用 rel="nofollow sponsored noopener"。
- 禁止虚假评分、虚假评论、虚假库存、虚假价格。
- 禁止使用 Product 结构化数据模拟不存在的商品信息。

## 内容定位
对外定位为：AI 工具选型、使用教程、订阅建议、办公/论文/编程效率方案。不要把网站包装成高风险交易站。

## 代码要求
- TypeScript 严格模式。
- 组件化开发。
- 所有 URL 从统一配置生成。
- 所有 CTA 从统一组件生成。
- 所有结构化数据从统一工具函数生成。
- 提交前运行 npm run build。
- 构建失败必须修复。

## 完成标准
- 首页可访问。
- 40 篇文章可访问。
- sitemap.xml 可访问。
- robots.txt 可访问。
- 每篇文章有唯一 title、description、canonical、H1、FAQ、CTA。
- 所有购买按钮跳转到小店。
- npm run build 通过。
- npm run check:seo 通过。
